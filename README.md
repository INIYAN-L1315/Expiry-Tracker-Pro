# Expiry Tracker Pro 🍎

A full-stack inventory management system designed to track product shelf-life and reduce waste through data-driven insights.

![Dashboard Demo](demo.png)

## 📝 Overview
Expiry Tracker Pro allows users to efficiently log inventory, monitor expiry dates, and visualize wastage patterns. This project demonstrates end-to-end full-stack development, bridging the gap between a relational PostgreSQL database and a dynamic JavaScript frontend.

## 🚀 Key Features
* **Inventory Management:** Seamlessly add, track, and manage product lifecycles.
* **Analytics Dashboard:** Visualize expiry trends using interactive charts (Chart.js).
* **Robust Backend:** Secure RESTful API built with Node.js and Express.
* **Data Integrity:** PostgreSQL-backed persistence with parameterized queries to ensure security.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla), Chart.js
* **Backend:** Node.js, Express.js
* **Database:** PostgreSQL
* **Tools:** VS Code, Git, NPM

## 📂 Project Structure
* `/public`: Frontend assets (HTML, CSS, JS)
* `/routes`: API endpoint logic
* `/database`: Database schema and configuration
* `server.js`: Main application entry point
